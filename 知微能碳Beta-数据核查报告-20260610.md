一、概述
================================================================================
  本次核查针对 Beta 版本进行全面页面数据检查，发现并修复了以下问题：

  1. 硬编码日期：13个Vue文件共20处日期写死为 2026-05-01~05-07
  2. 字段名不匹配：前端字段名与 API 返回字段名不一致（7处）
  3. Vue 渲染时序 Bug：nextTick 在 loading=true 时执行，DOM 未就绪（4处）
  4. API 数据查询错误：硬编码名称映射导致查询不到数据（3处）
  5. 数据生成遗漏：reset_demo.py 漏参数（3处）
  6. 页面显示逻辑：表格列名不适用、优先级顺序错误（2处）

  总计修复：约 30+ 处问题，涉及 25+ 个文件

2.3 Vue 渲染时序 Bug 修复（4处）
─────────────────────────────────────────
  根因: nextTick() 在 try 块内执行，此时 loading=true，v-else DOM 未渲染
  修复: nextTick() + render() 移到 finally 块之后

  | 文件 | 修复前 | 修复后 |
  |------|--------|--------|
  | EnergyStructure.vue | try{...nextTick();render()}finally{loading=false} | try{}finally{loading=false} nextTick();render() |
  | EnergyOverview.vue | 同上 | 同上 |
  | EnergyTrend.vue | 同上 | 同上 |
  | DeviceMonitor.vue | 同上 | 同上 |
  | ConsumptionAnalysis.vue | 同上 | 同上 |
  | Benchmarking.vue | nextTick(()=>{...}) 回调 | await nextTick(); ... |
  | EfficiencyAnalysis.vue | nextTick(()=>{...}) 回调 | await nextTick(); ... |
